# **App Name**: Petersthal Village Hub

## Core Features:

- Homepage with Welcome Message: Display a friendly welcome message, and village information on the homepage.
- Event Calendar: Show upcoming events in a monthly calendar view and as a list.
- Bus Connections and Directions: Integrate a map and bus schedules to show transportation options.
- Restaurant Listings: List local restaurants with descriptions and a map view.
- Accommodation Listings: Display a grid of accommodation options. This will not retrieve live data.
- Dynamic Club Pages: Generate pages for village clubs (TSV, KLJB, Trachtenverein, etc.) based on markdown content. Display image, club info and contact details on each page.
- Event Suggestion Tool: Generate event descriptions, that include date, time and location, using user-provided input. The tool will decide what content will be useful to include from the available context.

## Style Guidelines:

- Primary color: Soft, warm green (#90EE90) to evoke nature and friendliness.
- Background color: Light, desaturated green (#F0FFF0) to maintain a bright and airy feel.
- Accent color: Gentle yellow (#F0E68C) for highlights and interactive elements.
- Body and headline font: 'PT Sans' sans-serif for a modern, welcoming look.
- Use clean, simple icons that are easy to understand and visually appealing.
- Employ rounded cards and a responsive grid layout for a friendly and accessible design.
- Incorporate subtle transitions and animations for a smooth user experience.
